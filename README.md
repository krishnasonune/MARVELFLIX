
#  MARVELFLIX
It is a Netflix Clone with only collection of marvel movies

## Technologies Used

created this project using Technologies: -

- Angular
- Node.js
- Asp.net core web api 
- EntityFrame work core
- Mssql Database
- Azure ( for project deployment )
## Features

- Light weight WebApp
- Easy to use
- Simple GUI
- Responsive
